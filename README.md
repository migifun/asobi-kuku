# くくチャレンジ

子ども向けの九九（かけ算）学習ブラウザゲームです。

## 遊び方

1. 段を選ぶ（1〜9の段、または「ぜんぶランダム」）
2. 4つの選択肢から答えを選ぶ
3. 5問正解でクリア！アイテムがもらえる
4. 全30種のアイテムを集めよう

## 特徴

- 伝統的な九九の読み仮名つき（例: にさんがろく）
- 3秒後に答えの読みをヒント表示
- 間違えてもゲームオーバーにならない
- アイテムコレクション要素（全30種）
- 効果音つき（Web Audio API、音声ファイル不要）
- スマホ・タブレット対応
- PWA対応（ホーム画面に追加してオフラインでも動作）

## 開発

```bash
npm install
npm run dev
```

## 技術スタック

- Vite + TypeScript
- PWA（Service Worker + Web App Manifest）
- Web Audio API（効果音）
- localStorage（アイテム保存）
